## 核心愿景与演进哲学

本系统并非单纯的自动化工具集，而是一个以**"认知垂直整合"**为核心的个人外脑体系。其终极目标是通过高保真的原生行为捕捉与日常口语语料积累，在底层逐步沉淀出饱满、真实的数字画像，最终演化为能够替代主体进行前沿试错、经验吸收与自我思辨的**"数字代理（Digital Proxy）"**。

### 外挂硬盘高可动性原则

系统放弃 Mac 本机 `~/` 家目录依赖，将所有配置文件、数据库、扩展脚本以及"系统宪法种子"完全闭环在外挂硬盘路径中。实现**"换机即用，拔盘即走，思维痕迹永不丢失"**。

### 文档信息密度原则（可迁移性保障）

每次功能迭代后，必须自动更新涉及的 PRD 文档、Command_Reference 命令速查、以及 Pom 文档本身。确保每份文档的信息密度足以让系统在新环境中零上下文快速复现——包括准确的路径、数据结构、配置字段、以及功能边界。不等待用户提醒。

### 2.2 双脑协作桥梁 (Dual-Brain Bridge) — v2.2.0 新增

```
/Volumes/13384923891/bridge/
├── bridge.py                       # 加权上下文管理引擎（时间衰减 + 重要性权重）
├── shared_context.md               # 自动生成的共享上下文（双方 Agent 读取）
├── hermes_profile.md               # Hermes 身份档案（供小龙虾读取）
├── openclaw_profile.md             # 小龙虾身份档案（供 Hermes 读取）
├── logs/
│   └── bridge.db                   # SQLite 事件数据库
└── archive/                        # 过期条目归档目录
```

**设计原理：** 两个 AI 代理（Hermes + OpenClaw 小龙虾）通过桥梁共享状态。每条事件按「重要性 × 时间衰减」计算权重，系统级事件永不衰减，普通任务随时间减半，超出上下文上限时自动裁剪低权重条目。

**权重衰减规则：**
| 级别 | 基础权重 | 半衰期 | 说明 |
|------|---------|--------|------|
| 🏛️ system | 100 | 永不衰减 | 系统级配置/架构变更 |
| 🔴 critical | 80 | 90天 | 关键任务/安全事件 |
| 🟠 high | 50 | 30天 | 重要任务/主要功能 |
| 🟡 medium | 25 | 14天 | 一般任务 |
| ⚪ low | 10 | 7天 | 琐碎操作 |

**双方角色分工：**
- **Hermes (☤)** = 大脑：深度推理 + 代码执行 + 复杂任务编排
- **小龙虾 (🦞)** = 前哨：多渠道消息接收 + 初步过滤 + 轻量级回复

```bash

alias ctx="bash \$HERMES_DRIVE/context-indicator once"

chmod +x $HERMES_DRIVE/scripts/* 2>/dev/null
```

执行 `source ~/.zshrc` 使别名立即生效。

## 四、 新功能规范：02_灵感白板层 (Idea Whiteboard)

1. **零延迟捕获：** 输入命令时，系统必须在 50ms 内响应完成，绝对禁止在外挂硬盘 IO 期间发起大模型网络请求，确保编程主线程零中断。
2. **异步语义缝合 (flush 机制)：** 当用户执行 `ctx-idea flush` 时，Hermes 提取外挂盘数据库中所有 pending 的想法，结合盘内 `ctx-flow` 的执行轨迹进行交叉对比。
3. **输出落盘：** 在 `/Volumes/13384923891/hermes-agent/context-system/Whiteboard/` 路径下自动生成以日期命名的缝合报告（如 `Whiteboard_Log_20260521.md`）。

## 五、 未来演进：系统孵化舱（种子库设计）

此部分属于"系统种子"，目前技术处于积蓄期，Hermes 需将其作为长期架构规范留存，并在后续对话中以此作为系统大进化的大方向支撑。

### 1. 03_交互渲染层 (Visual Parser) — 降噪种子

设计一个正则高亮渲染挂载器。消除终端大段英文与复杂代码堆砌带来的视觉疲劳。对核心逻辑、关键变量进行颜色级别（Color-coded）的区分。

### 2. 04_语言镜像层 (Language Corpus) — 反思种子

拒绝流于表面、客套的虚假角色扮演。在后台静默收集语音转换后的原始文本，分析高频出现的低效率表达、逻辑断层，逆向生成极具针对性的表达训练提示（就像一个暗箱监察的语言表达教练）。

### 3. 05_数字代理层 (Digital Proxy) — 终极形态种子

基于盘内的行为轨迹、思维流、以及真实口语表达特征，三合一坍缩出高度精准、客观、不带情感偏见的个人底层画像。创造一个"约等于我"的数字人，替我前置探索、去犯错、去踩坑，并以我最容易领悟的方向将经验反馈给我。
